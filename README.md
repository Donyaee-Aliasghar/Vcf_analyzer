## Installation

```bash
https://github.com/Donyaee-Aliasghar/Vcf_analyzer.git
```

```python
poetry install
```

## Usage
1.After cloning go to the Vcf_analyzer
```bash
cd Vcf_analyzer
```
2.Run the following code
```python
python -i <input vsc file(.vcf or .vcf.gz)> cli.py
```
